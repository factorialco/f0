# What is F0

F0 is Factorial's **design system** — a living ecosystem, not just a component library.

## What it is

- A **shared component library** (`@factorialco/f0-react`).
- A set of **design rules and tokens** — colors, spacing, typography.
- A **contribution lifecycle** anyone in Factorial can use.
- **Owned by the Foundations team, built with the whole company.**

## Why it helps

- **One source of truth for UI** — no more five different buttons.
- **Accessibility built in**, so every product gets it.
- **Faster shipping** — compose features, don't rebuild primitives.
- **A predictable look** across the whole Factorial product.

## What it is _not_

- **Not a product-specific UI kit** — if only your product uses it, it doesn't belong here.
- **Not a sandbox to iterate styles** — once stable, a component has a contract.
- **Not a Figma library** — the repo is the source of truth.
- **Not owned by Foundations alone** — anyone can propose and contribute.

## Where F0 lives

A few surfaces make up F0 — and the **repo** is the source of truth everything else reads from:

- **Your AI agent** — the main entry point. F0 ships **AGENTS.md** and skills in the repo, so any AGENTS.md-aware agent (Claude Code, Codex, OpenCode, …) reads the catalogue and rules and builds for you.
- **Storybook** — the visual catalogue of every component, with live examples and when-to-use docs. → [f0.factorial.dev ↗](https://f0.factorial.dev)
- **F0 Compose** — a canvas that renders the composition an agent builds, with real F0 pieces (work in progress). → [factorial-composer ↗](https://github.com/factorialco/factorial-composer)
- **The repo (`f0`)** — the **source of truth**: components, rules, skills and docs. Everything else is a view over it. → [GitHub ↗](https://github.com/factorialco/f0)
- **#f0-support (Slack)** — where every request starts (contributions, bugs, promotions): you raise a message and Foundations picks it up. → [Open #f0-support ↗](https://factorialteam.slack.com/archives/C082ZNKS403)
- **Figma library** — an optional sandbox to sketch variants (a Composer prototype works too). Not the source of truth — the code is. → [F0 library in Figma ↗](https://www.figma.com/files/1209519141454845458/team/1359486354120896816)
