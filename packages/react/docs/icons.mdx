# Icons

F0 ships three sets of icons, all generated from SVG sources in `packages/core/assets/icons`.

| Set          | Import path                            | When to use                                                                                                                        |
| ------------ | -------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| **App**      | `@factorialco/f0-react/icons/app`      | General UI actions and concepts                                                                                                    |
| **Module**   | `@factorialco/f0-react/icons/modules`  | Factorial product modules — prefer <LinkTo kind="Avatars/AvatarModule" story="Documentation">F0AvatarModule</LinkTo> for rendering |
| **Animated** | `@factorialco/f0-react/icons/animated` | Motion-enhanced icons, trigger animation via `state="animate"`                                                                     |

### Usage

Render any icon with <LinkTo kind="Icon" story="Documentation">F0Icon</LinkTo> for inline use, or <LinkTo kind="Avatars/AvatarIcon" story="Documentation">F0AvatarIcon</LinkTo> for the contained avatar treatment:

```tsx
import { F0Icon } from "@factorialco/f0-react"
import { Pencil } from "@factorialco/f0-react/icons/app"

return <F0Icon icon={Pencil} size="md" />
```

```tsx
import { F0AvatarIcon } from "@factorialco/f0-react"
import { Pencil } from "@factorialco/f0-react/icons/app"

return <F0AvatarIcon icon={Pencil} size="md" />
```

For module icons, use <LinkTo kind="Avatars/AvatarModule" story="Documentation">F0AvatarModule</LinkTo> with the module ID — do not render the raw SVG directly:

```tsx
import { F0AvatarModule } from "@factorialco/f0-react"

return <F0AvatarModule module="timeoff" size="md" />
```

For animated icons, pass `state="animate"` on hover:

```tsx
import { F0Icon } from "@factorialco/f0-react"
import { SomeAnimatedIcon } from "@factorialco/f0-react/icons/animated"

return (
  <F0Icon icon={SomeAnimatedIcon} state={isHovered ? "animate" : "normal"} />
)
```

> Want to add a new icon? See the <LinkTo kind="Foundations/Icons/Add a new icon" story="Documentation">step-by-step guide</LinkTo>.
