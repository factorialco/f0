# Spacing

Factorial One uses the
[default Tailwind spacing scale](https://tailwindcss.com/docs/customizing-spacing#default-spacing-scale)
based on 4px units. The names are on a numerical scale from 0 to 96.

We define two scales for spacing:

- absolute scale with sizes in _pixels_;
- relative scale with _rem_;

We want to scale our UI when user changes font settings, so all the containers
must adapt the font size. Relative scale is used for block size properties, like
`width`, `height`, `minWidth`, `maxWidth`, `minHeight`, `maxHeight`, `size`,
`flexBasis` and `textIndent`.

The rest of the properties use pixels: `padding`, `margin`, `gap`, `space`,
`translate`, `scrollMargin`, and `scrollPadding`.
