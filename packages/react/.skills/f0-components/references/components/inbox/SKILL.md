## Overview
The Inbox component suite provides a structured way to display chronological activities, notifications, and approval workflows. It is designed to handle complex metadata like timestamps, categories, and multi-step approval statuses.

## Sub-components
- **Activity**: A container for a list of activity items, often used for feeds.
- **ActivityItem**: The individual unit of a feed, displaying a title, description, and metadata.
- **ApprovalHistory**: A specialized view for tracking the lifecycle of requests (e.g., Pending, Approved, Rejected).
