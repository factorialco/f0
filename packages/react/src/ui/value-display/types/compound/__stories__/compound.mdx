### compound

Renders a compound value made of multiple segments separated by a delimiter. Each
segment can have its own semantic tone.

#### Value type

```ts
type value = {
  segments: CompoundSegment[]
  separator?: string // default "/"
}

type CompoundSegment =
  | {
      type: "text"
      value: string | number | undefined
      placeholder?: string
      tone?: CompoundTone
    }
  | {
      type: "number"
      value: number | undefined
      units?: string
      unitsPosition?: "left" | "right"
      decimalPlaces?: number
      placeholder?: string
      tone?: CompoundTone
    }
  | {
      type: "amount"
      value: number | undefined
      currency?: {
        symbol: string
        symbolPosition?: "left" | "right"
        decimalPlaces: number
      }
      placeholder?: string
      tone?: CompoundTone
    }
```
