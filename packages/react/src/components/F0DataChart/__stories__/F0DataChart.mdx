# F0DataChart

Unified chart component. Uses a discriminated `type` prop to select the chart variant.

```tsx
import { F0DataChart } from "@factorialco/f0-react"
```

## Quick Reference

| Type | Best for | Key props |
|------|----------|-----------|
| `bar` | Comparing categories or time periods | `series`, `categories`, `orientation`, `stacked` |
| `line` | Trends over time | `series`, `categories`, `lineType`, `showArea`, `showDots` |
| `funnel` | Conversion pipelines, hiring stages | `series`, `sort`, `orient`, `showConversion`, `colorScale` |
| `pie` | Part-of-whole distributions | `series`, `innerRadius`, `showPercentage` |
| `radar` | Multi-dimensional comparisons | `series`, `indicators`, `showArea` |
| `gauge` | Single KPI / progress indicator | `value`, `min`, `max`, `name` |
| `heatmap` | Density across two dimensions | `data`, `xCategories`, `yCategories`, `showLabels` |

## Customization

### Value & Category Formatters

All chart types support `valueFormatter` to customize how numbers are displayed in axes, labels, and tooltips. Bar and line charts also support `categoryFormatter` for axis labels.

```tsx
<F0DataChart
  type="bar"
  valueFormatter={(v) => `${v / 1_000_000}M €`}
  categoryFormatter={(v) => v.slice(0, 3)}
  // ...
/>
```

### ECharts Escape Hatch

Bar and line chart types accept an `echartsOptions` prop for advanced customization. Options are shallow-merged on top of the generated config.

```tsx
<F0DataChart
  type="bar"
  echartsOptions={{
    animation: false,
    grid: { left: 80 },
  }}
  // ...
/>
```

### Color Tokens

Use F0 design tokens for custom colors: `"malibu"`, `"purple"`, `"grass"`, `"orange"`, `"red"`, `"viridian"`, `"barbie"`, `"lilac"`, `"smoke"`, `"army"`, `"flubber"`, `"indigo"`, `"camel"`, `"radical"`, `"yellow"`.
