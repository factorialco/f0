# FilterPicker Internal

The FilterPicker component exposes the different parts of the component to
customize the layout for internal use.

```tsx
import * as OneFilterPicker from "@/components/OneFilterPicker"

return (
  <OneFilterPicker.Root filters={filters} value={value} onChange={onChange} ...>
    <OneFilterPicker.Controls />
    <OneFilterPicker.Presets />
    <OneFilterPicker.ChipsList />
  </OneFilterPicker.Root>
)
```
