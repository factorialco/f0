# Data collection Full height

## Introduction

Having in the same viewport the filters, the data collection data and the
pagination is a very useful and common pattern.

To achive this, `OneDataCollection` provides you the `fullHeight` property. When
it is true the table will use `height: 100%` and the content will try to use the
maximum height possible.

> Remember the `height: 100%` is relative to the parent container so if you want
> to limit the height of the data collection you must set the correct height to
> the parent container.

### Example

```tsx
<div className="flex h-full flex-col">
  <div>Other page content</div>
  <div className="min-h-0 flex-1">
    <OneDataCollection
      source={source}
      visualizations={visualizations}
      fullHeight
    />
  </div>
  <div>Other page content</div>
</div>
```

Here the data page will use the maximum height possible and the
OneDataCollection component will use the 100% of that wrapper.
