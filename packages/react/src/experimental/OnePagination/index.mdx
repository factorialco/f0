# Pagination

Divides content into separate pages, allowing users to navigate through large
sets of data or items one page at a time.

```tsx
import { OnePagination } from "@factorialco/f0/dist/experimental"
```

### Indeterminate

If the number of pages cannot be accurately determined, set the `totalPages`
prop to `0`. This will remove the button numbers and rely solely on the arrow
icon buttons to navigate.

The `hasNextPage` prop is used to indicate if there are more pages available.
