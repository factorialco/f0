# DatePicker Presets

## Introduction

Presets are predefined date ranges and granularity that can be used to select a
date range easily and with semantic meaning. e.g. Today, Last 7 days, Last 30
days, Last month, Last 3 months, Last 6 months, Last year, etc.

## Usage

```tsx
const presets: DatePreset[] = [
  {
    label: "Today",
    granularity: "day",
    value: granularityDefinitions.day.toRange(new Date()),
  },
  {
    label: "Last 7 days",
    granularity: "day",
    value: () =>
      granularityDefinitions.day.toRange(subDays(new Date(), 7), new Date()),
  },
]
<OneDatePicker presets={presets} />
```

## Predefined presets

The `OneDatePicker` component comes with a set of predefined presets that can be
used to select a date range easily and with semantic meaning. e.g. Today, Last 7
days, Last 30 days, Last month, Last 3 months, Last 6 months, Last year, etc.

The predefined presets are:

- Today
- Yesterday
- Last 7 days
- This week
- Last week
- This month
- Last month
- Last 3 months
- Last 6 months
- This quarter
- Last quarter
- This half year
- Last half year
- Last year
- Last 3 years

> Check the `packages/react/src/experimental/OneDatePicker/presets.ts` file for
> the complete list of predefined presets.

### Using the predefined presets

The predefined presets are used by setting the `presets` prop to the array of
presets.

```tsx
import { presets as predefinedPresets } from "@factorialco/factorial-one/react/experimental/OneDatePicker"

const presets: DatePreset[] = [
  {
    label: "Today",
    granularity: "day",
    value: granularityDefinitions.day.toRange(new Date()),
  },
  predefinedPresets.last7Days,
  predefinedPresets.last30Days,
  predefinedPresets.lastMonth,
  predefinedPresets.last3Months,
  predefinedPresets.last6Months,
  predefinedPresets.lastYear,
  predefinedPresets.last3Years,
]
;<OneDatePicker presets={presets} />
```

As you can see, the predefined presets can be used together with custom presets.
