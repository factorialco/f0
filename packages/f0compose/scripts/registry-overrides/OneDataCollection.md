## Description

The flagship collection pattern: tables, lists, kanbans and grids over a typed data source. Supports filters, sorting, pagination, item actions, bulk actions, search, visualization toggles, empty states.

## Notes

For full filter/sort/pagination behavior, read `packages/react/src/patterns/OneDataCollection/` and the stories under `__stories__/`. For prototypes, prefer the canonical "table with simple columns + ItemActions" shape.
